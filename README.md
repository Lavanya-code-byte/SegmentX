# 📊 SegmentX – AI-Powered Customer Segmentation Dashboard

SegmentX is an intelligent customer analytics dashboard designed to perform customer segmentation using RFM (Recency, Frequency, Monetary) analysis. It helps businesses identify customer behavior patterns and classify customers into meaningful groups for better marketing and decision-making.

The application generates interactive visualizations and analytics reports to help understand customer engagement, revenue contribution, and customer retention.

---

## 🚀 Features

- Customer segmentation using RFM analysis
- Interactive dashboard with charts and analytics
- Revenue and customer insights visualization
- Customer behavior tracking
- Real-time segment distribution analysis
- Responsive and modern UI
- Customer performance metrics

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Lavanya-code-byte/SegmentX.git
cd SegmentX
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

### 4. Open in Browser

```text
http://localhost:5173
```
